# Arduino 기반 전자부품 자동 검사·분류 시스템

컨베이어를 따라 이동하는 전자부품을 센서로 감지하고, PC의 검사 결과에 따라 서보모터로 분류하는 스마트 제조 하드웨어 제어 프로젝트입니다.

Arduino MEGA 2560을 중심으로 컨베이어 모터, 적외선 센서, 서보모터, 비상정지 버튼, 부저를 제어했습니다.  
PC와는 시리얼 통신을 사용해 시작·정지 명령과 분류 명령을 주고받도록 구현했습니다.

## 주요 기능

- 적외선 센서를 이용한 전자부품 감지
- 감지 후 컨베이어 일정 시간 정지
- PC 명령에 따른 컨베이어 시작 및 정지
- 검사 결과에 따른 서보모터 분류 동작
- 비상정지 버튼 입력 시 모터 정지 및 부저 경고
- PC 연결 상태 확인을 위한 주기적 통신
- PC 통신이 끊기면 컨베이어 자동 정지
- DHT22 센서를 이용한 온도값 전송
- 워치독 타이머를 이용한 시스템 멈춤 방지

---

## 동작 순서

1. PC에서 Arduino로 시작 명령을 전송합니다.
2. 컨베이어가 전자부품을 검사 위치까지 이동시킵니다.
3. 적외선 센서가 전자부품을 감지하면 컨베이어가 잠시 정지합니다.
4. PC에서 검사 결과를 Arduino로 전송합니다.
5. 검사 결과에 맞는 서보모터가 동작해 전자부품을 분류합니다.
6. 서보모터가 원래 위치로 돌아오고 다음 부품 검사를 진행합니다.
7. 비상정지 버튼이 눌리거나 PC 통신이 끊기면 컨베이어를 정지합니다.

## PC 명령 형식

PC에서는 JSON 형식으로 명령을 보낼 수 있습니다.

### 컨베이어 시작

```json
{"cmd":"start","seq":1}
```

### 컨베이어 정지

```json
{"cmd":"stop","seq":2}
```

### 모터 상태 변경

```json
{"cmd":"motor","value":1,"seq":3}
```

- `value: 1` → 동작
- `value: 0` → 정지

### 서보모터 동작

```json
{"cmd":"servo","target":"A","value":110,"seq":4}
```

- `target: "A"` → 서보모터 1
- `target: "B"` → 서보모터 2
- `value` → 동작 각도

### 연결 상태 확인

```json
{"cmd":"ping","seq":5}
```

Arduino는 명령을 정상적으로 받으면 다음과 같이 응답합니다.

```json
{"evt":"ack","seq":5}
```

## Arduino에서 보내는 데이터

### 부팅 알림

```json
{"evt":"boot","fw_version":"1.2.0"}
```

### 주기적 연결 확인

```json
{"evt":"hb","uptime":1000,"seq":1}
```

### 센서 상태

```json
{"evt":"sensor","st":1,"t":24.5,"s6":0,"seq":2}
```

| 항목 | 의미 |
|---|---|
| `st` | 시스템 동작 상태 |
| `t` | 측정 온도 |
| `s6` | 물체 감지 상태 |
| `seq` | 데이터 순서 번호 |
